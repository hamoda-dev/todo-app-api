## Todo App Api
this is simple todo app api

### Installation
1 - clone project
```
    git clone https://github.com/hamoda-dev/todo-app-api.git
```

2 - install dependency
```composer log
    composer install
```

3 - setup app
```composer log
    php artisan migrate --seed
```

4 - create admin
```composer log
    php artisan todo:create-admin
```
